# Spotify Profile App

A wep app to see your spotify profile in detail.
Feature to see your
  - Top Artists
    - All time
    - Last Six Months
    - Last four weeks
  - Top Tracks of all time
    - All time
    - Last Six Months
    - Last four weeks
  - Recently Played tracks
  - Playlists
